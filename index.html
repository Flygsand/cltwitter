cltwitter

cltwitter is a command-line utility for posting updates to Twitter.

Example usage

cltwitter can handle two sorts of input. Either a command line argument: 

$ tweet "Hello Twitter!"
or from standard input (here piping from the UNIX command fortune: 
$ fortune | tweet

Get it

  • Source tarball
  • Windows binary (x86 32-bit)
